## Project setup(初始化init)
```
---use npm setup
npm install
---use yarn setup
(npm install -g yarn)
yarn install
yarn serve
```

### Compiles and hot-reloads for development(启动服务器-本地)
```
---use npm start
npm run serve
---use yarn start
yarn serve
```

### Compiles and minifies for production(生成生产环境文件-配置后端使用)
```
npm run build or download 'dist' file
```

### Lints and fixes files(修复文件不规则的地方 推荐使用自动修复)
```
npm run lint
```
